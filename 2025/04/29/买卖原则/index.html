✅ 买点五式（择时介入点）

买点 类型 特征与信号说明 操作建议
一号买点 突破启动买 首次放量突破平台或关键均线 小仓试盘，防止假突破
二号买点 回踩确认买 回踩关键支撑（如平台或均线）获得支撑不破 加仓，确认突破有效
三号买点 放量加速买 再次放量突破，趋势确认加速 加仓/追涨，吃主升浪
四号买点 震荡区反弹买 区间低位缩量止跌、反弹信号出现 快进快出，轻仓操作
五号买点 洗盘重启买 主力洗盘完成，缩量回踩后再次放量启动 潜伏中线行情，可中仓

❌ 卖点三式（控制风险点）

卖点 类型 特征与信号说明 操作建议
一式 目标达成卖 达到自己设置的盈利目标（如+30%、+50%） 分批止盈，保护成果
二式 趋势破位卖 跌破重要支撑/趋势线、均线死叉、成交量放大 果断止损或减仓
三式 高位异动卖 高位放量大阴线、长上影、十字星，主力可能出货 快速止盈，防止被套

🎯 操作节奏图建议

  1. 低位布局：一号试盘 → 二号确认 → 三号加速
  2. 震荡阶段：四号快进快出，防止被套
  3. 洗盘结束后：五号买入潜伏中线
  4. 目标达成或风险来临：一式/二式/三式果断卖出

买点一式：一号买点（支撑回踩）

一、核心定义

一号买点是指股价 回踩趋势支撑位 后，出现企稳信号的买入机会。常用于上涨趋势中的“低吸”。

二、出现位置

  • 股价前期上涨后出现回调；
  • 回调到重要支撑位（比如趋势线、均线、箱体底部、前高等）；
  • 出现止跌信号，如：缩量阴线、十字星、企稳阳线、下影线等。

三、识别关键点

条件 判断标准
支撑位 5日/10日/20日均线、上升趋势线、箱体支撑、前高位置等
K线形态 下影线长、缩量止跌、阳包阴、十字星、红三兵等
成交量 回踩时缩量、止跌时有放量阳线更佳
大盘环境 同期大盘不大幅下跌，行业板块最好是活跃的

四、买入策略

  • 低吸原则：股价回踩支撑后企稳再介入，避免盲目接刀；
  • 分批买入：第一次试仓，后续回踩确认时可加仓；
  • 止损设置：跌破支撑位3%以上止损，防止假支撑。

买点二式：二号买点（突破回踩）

一、核心定义

二号买点是指股价突破关键压力位（如平台、前高、箱体等）后，回踩确认支撑并企稳，再次上涨的买入机会。

二、出现位置

  1. 前期平台或前高附近横盘震荡；
  2. 股价放量突破该区域；
  3. 后续缩量回踩，不破平台顶或前高；
  4. 出现止跌信号后反弹，即为买点。

三、识别要点

条件 判断标准
突破位置 前高、平台、箱体顶部
回踩表现 缩量、止跌阳线、十字星、下影线较长等
成交量 突破时放量、回踩时缩量，反弹再放量最佳
成交密集区 回踩不跌破之前的主力建仓密集区更稳健

四、形态描述

突破平台 → 回踩确认 → 企稳反弹
这种买点意味着前方筹码已经消化，主力拉升意图明确，是一个“确认突破”的信号。

五、买入策略

  • 等回踩完成后的止跌阳线（可配合成交量）；
  • 设好止损：回踩若跌破平台底部3%以上，止损退出；
  • 可用小仓试盘，回踩反弹再加仓。

买点三式：三号买点（强势中继）

一、核心定义

三号买点是指在主升浪趋势中，个股短暂整理后再次加速上涨的买点。
它常出现在突破之后的高位震荡区，类似“骑上牛背再加鞭”。

二、出现位置

  • 个股已有明显涨幅（进入主升段）；
  • 在上涨过程中出现短暂调整（高位横盘/小幅回踩）；
  • 短线洗盘结束，放量突破前高或平台，继续上攻。

三、识别要点

条件 判断标准
走势位置 处于主升浪中途，前期已拉升一段
整理形态 高位横盘、旗形整理、上升三角形、平台整理等
放量突破 整理结束后再度放量阳线突破前高
板块助力 热门题材或行业加持更有效
时间特征 整理不宜太长，一般3~7天为宜，否则变盘不确定性增加

四、图形形态描述

像一头牛冲上山坡后，在坡上小歇，然后继续加速冲顶！

常见整理形态如下：

  • 上升三角形突破
  • 旗形整理后上攻
  • 平台整理突破新高

五、买入策略

  • 确认突破：一定要等放量阳线或突破K线出现后再介入；
  • 防止假突破：若突破但未放量或回踩破位，立刻止损；
  • 仓位控制：因位置较高，适合顺势加仓或波段操作，不建议新建重仓。

买点四式：四号买点（尾声拉升）

一、核心定义

四号买点是指个股经过长时间上涨后，在趋势末期出现的最后一波主力拉升。通常是庄家诱多出货前的“冲顶波”。

二、出现位置

  • 股价已连续上涨数波；
  • 涨幅超过50%甚至100%；
  • 出现加速上涨、放量涨停；
  • 个股/板块情绪极度亢奋，媒体热炒；
  • 盘面极强但出现明显顶部特征之前的一瞬间。

三、识别要点

条件 判断标准
涨幅 通常前期已翻倍或连续大涨
加速特征 短期内连续放量涨停或阳线，缩量横盘后突然涨停
情绪面 舆论、股吧、财经新闻等高度关注该股或板块
技术形态 5日均线远离10日/20日均线，K线角度接近垂直上攻
常见陷阱 拉高后迅速放量跳水，诱多抛盘式出货

四、图形形态描述

类似“冲天牛”状态——牛已经冲到悬崖边，最后几步还在狂奔。但一旦踩空，跌得也快。

  • 通常形态如：急涨 + 高位整理 + 最后一根拉升阳线
  • 有时是高位涨停或盘中快速冲高

五、买入策略

  • 必须快进快出：适合做短线博弈、不能恋战；
  • 小仓位参与：仅限于情绪极强热点，最多10~20%仓位试探；
  • 有收益就走：次日冲高就止盈，绝不贪；
  • 高风险预期：买之前必须设好止损，且要知道这可能是最后一波。

买点五式：五号买点（洗盘结束再起涨）

一、核心定义

五号买点是指主力洗盘结束、清洗浮筹之后的低位二次启动点，通常伴随恐慌杀跌或横盘阴跌后突然止跌反转，庄家完成换手、再次拉升。

二、出现位置

  • 前期已经涨过一波；
  • 中途经历一轮较大幅度调整（主动洗盘）；
  • 杀跌诱空后成交量萎缩，止跌回稳；
  • 突然放量转阳、长下影或涨停板，开启第二波行情。

三、识别要点

条件 判断标准
洗盘特征 跌破支撑、放量阴线 → 缩量止跌 → 放量反包
成交量 先大后小再放大，即“倒V → 缩 → 爆”结构
技术形态 双底、W底、头肩底、下影线、早晨之星等止跌K线形态
时间规律 洗盘一般持续3~10天，有时时间更长，需耐心等待止跌信号
情绪变化 恐慌砸盘 → 无人问津 → 突然爆量拉升

四、图形形态描述

像牛冲了一段后回头喘息，然后在谷底蓄力再次爆发

  • 类似一个“V”形反转或“W”底部
  • K线表现常为：一根下影长阳，或涨停板穿破整理平台

五、买入策略

  • 信号出现才进场：不要抄底，要等止跌+反包确认；
  • 建仓时机：可分两次建仓（第一根反包阳线+突破确认加仓）；
  • 仓位控制：若洗盘成功概率高，可适度重仓（如20~40%）；
  • 止损设置：若再次跌破平台低点，立刻止损退出；

🧨 典型卖点类型（卖点三式）

1️⃣ 卖点一式：目标达成卖

  • 涨幅已经较大，达到你的盈利预期，比如+30%、+50%，可逐步止盈。
  • 结合“321”建仓模型，逐步分批出货。

💡 举例：买入价10元，涨到13元或15元（30%-50%），开始减仓止盈。

2️⃣ 卖点二式：趋势破位卖

  • 股价跌破重要支撑线（如上升趋势线、20日线、平台支撑）。
  • 成交量放大 + 跌破支撑，需果断卖出。

💡 特别是破位后反抽不能重新站回支撑线，确认趋势反转，更需卖出！

3️⃣ 卖点三式：高位放量异动卖

  • 高位大阴线 / 放量长上影线 / 巨量十字星，暗示主力出货。
  • 结合盘口异动，尤其出现突然放量大跌或大单卖出时。

💡 注意！高位滞涨 + 异动放量是典型“主力出货”信号，不能恋战！

🧭 二、补充卖点判断辅助指标

  • MACD 死叉、背离
  • KDJ 高位钝化并下拐
  • RSI 超买（>80）开始回落
  • 高位“双头”“三顶”形态
  • 产业周期进入秋末 → 冬阶段
